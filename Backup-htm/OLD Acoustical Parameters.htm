Acoustical Parameters (ISO 3382) - new version

This module is still in Alpha stage. It allows for computation of ISO3382 parameters in 1/3 octave bands, and it also computes Ando's binaural parameters (IACC) and ISO3382 lateral fractions (LE, LF, LFC).
In the first case, a binaural impulse response must be processed. In the second case, the module expects that the impulse response contains Omni on the left and Figure-of-Eight on the right channel. This can easily be obtained using the channels W and Y of a Soundfield microphone.
The actual version only works with waveforms at 44.1 kHz, and often crashes...

It will be finished in the next months.
